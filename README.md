# Пользовательская библиотека C++ контейнера Vector и линейной алгебры

Легковесная, шаблонная, безопасная относительно исключений библиотека контейнеров (переосмысление базового функционала `std::vector`), оптимизированная для задач линейной алгебры. Проект разработан с использованием стандартов C++17 и демонстрирует надежное управление памятью, строгую гарантию безопасности исключений и бесшовную интеграцию с алгоритмами STL.
 
---

## Ключевые особенности

* **Универсальный шаблонный контейнер**: Поддерживает любые типы (`template <typename T>`) с полностью настраиваемым динамическим изменением размера массива.
* **Амортизированная вставка O(1)**: Кастомная стратегия управления памятью с удвоением вместимости (`reserve`) для достижения оптимальной амортизированной производительности при добавлении элементов.
* **Строгая гарантия безопасности исключений**: Надежная обработка ошибок достигается за счет реализации идиомы **Copy-and-Swap** в операторе присваивания (`operator=`).
* **Встроенные операции линейной алгебры**:
    * Сложение векторов (`+`, `+=`)
    * Вычитание векторов (`-`)
    * Скалярное произведение (`*`) с валидацией размерностей (генерирует исключение `std::invalid_argument` при несовпадении).
* **Совместимость с STL**: Прямая совместимость с алгоритмами стандартной библиотеки C++ (например, `std::accumulate`, `std::sort`, `std::copy`) благодаря использованию сырых указателей в качестве итераторов произвольного доступа (`begin()` и `end()`).
* **Современная система сборки CMake**: Полная интеграция с современной конфигурацией CMake для кроссплатформенной сборки и простой настройки в IDE.

## Инструкции по сборке и запуску

### Требования
* Компилятор с поддержкой C++17 (MSVC 2017+, GCC 7+, Clang 5+)
* **CMake** (версия 3.10 или выше)

### Вариант 1: Использование Visual Studio (Рекомендуется для Windows)
1. Запустите **Visual Studio 2019/2022**.
2. Выберите **Открыть локальную папку (Open a local folder)** и укажите корневую директорию `LinearAlgebraVector`.
3. Visual Studio автоматически настроит кэш CMake.
4. Выберите `VectorApp.exe` в качестве стартового проекта и нажмите кнопку **Запуск/Отладка (Run/Debug)** (зеленый треугольник).

### Вариант 2: Командная строка (Кроссплатформенный подход)
Из корневой директории проекта выполните следующие команды:

```bash
# Создание и переход в директорию сборки
mkdir build && cd build

# Конфигурация проекта
cmake ..

# Сборка исполняемого файла
cmake --build .

# Запуск приложения
./VectorApp
```

## Быстрый старт и пример использования

Ниже приведен короткий пример, демонстрирующий, как инициализировать векторы, выполнять математические операции и интегрировать библиотеку со стандартными алгоритмами:

```cpp
#include <iostream>
#include <numeric>
#include "Vector.h"

int main() {
    // 1. Инициализация и STL итераторы
    Vector<int> v;
    v.push_back(10);
    v.push_back(20);
    v.push_back(30);

    // Простое суммирование элементов с помощью std::accumulate
    int sum_stl = std::accumulate(v.begin(), v.end(), 0);
    std::cout << "Сумма через STL: " << sum_stl << "\n"; // Выведет 60

    // 2. Операции линейной алгебры
    Vector<double> a(3);
    a[0] = 1.0; a[1] = 2.0; a[2] = 3.0;

    Vector<double> b(3);
    b[0] = 4.0; b[1] = 5.0; b[2] = 6.0;

    // Сложение векторов
    Vector<double> sum = a + b; // [5.0, 7.0, 9.0]
    std::cout << "a + b = " << sum << "\n";

    // Скалярное произведение
    double dot = a * b; // 1*4 + 2*5 + 3*6 = 32
    std::cout << "a * b = " << dot << "\n";

    return 0;
}
```
