# Пользовательская библиотека C++ контейнера Vector и линейной алгебры

Легковесная, шаблонная, безопасная относительно исключений библиотека контейнеров (переосмысление базового функционала `std::vector`), оптимизированная для задач линейной алгебры. Проект разработан с использованием стандартов C++17 и демонстрирует надежное управление памятью, строгую гарантию безопасности исключений и бесшовную интеграцию с алгоритмами STL.

---

## 🚀 Ключевые особенности

* **Универсальный шаблонный контейнер**: Поддерживает любые типы (`template <typename T>`) с полностью настраиваемым динамическим изменением размера массива.
* **Амортизированная вставка O(1)**: Кастомная стратегия управления памятью с удвоением вместимости (`reserve`) для достижения оптимальной амортизированной производительности при добавлении элементов.
* **Строгая гарантия безопасности исключений**: Надежная обработка ошибок достигается за счет реализации идиомы **Copy-and-Swap** в операторе присваивания (`operator=`).
* **Встроенные операции линейной алгебры**:
    * Сложение векторов (`+`, `+=`)
    * Вычитание векторов (`-`)
    * Скалярное произведение (`*`) с валидацией размерностей (генерирует исключение `std::invalid_argument` при несовпадении).
* **Совместимость с STL**: Прямая совместимость с алгоритмами стандартной библиотеки C++ (например, `std::accumulate`, `std::sort`, `std::copy`) благодаря использованию сырых указателей в качестве итераторов произвольного доступа (`begin()` и `end()`).
* **Современная система сборки CMake**: Полная интеграция с современной конфигурацией CMake для кроссплатформенной сборки и простой настройки в IDE.
